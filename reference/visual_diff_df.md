# Get a widget that displays diff between data frame

Thin wrapper around
[`diffviewer::visual_diff()`](https://diffviewer.r-lib.org/reference/visual_diff.html).

## Usage

``` r
visual_diff_df(old, new, width = NULL, height = NULL)
```

## Arguments

- old, new:

  Pair of data frames to compare

- width, height:

  Output size

## Value

A HTML widget

## Details

Also writes to tempfiles to make sure it works

## Examples

``` r
mtcars2 <- mtcars

mtcars2[1, 1] <- NA
visual_diff_df(old = mtcars, new = mtcars2)
```
